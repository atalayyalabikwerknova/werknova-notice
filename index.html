WerkNova

WerkNova is a personal software project.

We are aware of reports describing fake job interviews and malware campaigns using the name "WerkNova".

Those activities are not related to this project.

WerkNova will never ask you to run terminal commands such as curl, bash, or PowerShell, or install software as part of a recruitment process.

If you receive such a request, treat it as suspicious and verify it independently.
