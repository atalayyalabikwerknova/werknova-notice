WerkNova

WerkNova is an independent software project.

We are aware of reports describing fake job interviews and malware campaigns using the name "WerkNova".

Those activities are not affiliated with this project.

We never ask candidates or users to:

  • run terminal commands such as curl, bash, or PowerShell
  • disable security protections
  • install software from unofficial sources

If someone asks you to do this during a recruitment process, treat it as suspicious and verify the request independently.
